# Fallen Angel RL/RP

Fallen Angel RL/RP is a multiplayer roleplay project currently built for **SA:MP (open.mp)** and **FiveM**.

## Vision

We are building a long-term roleplay universe across platforms.  
Current focus is SA:MP and FiveM, with planned expansion to **GTA 6** when the platform ecosystem is ready.

## About

This organization hosts official Fallen Angel RL/RP resources, including:

- server code and gameplay systems
- backend and database integrations
- infrastructure and deployment configuration
- internal and public project documentation

## Platforms

- SA:MP / open.mp
- FiveM
- GTA 6 (planned)

## Development Status

Active development with ongoing system improvements and feature expansion.
